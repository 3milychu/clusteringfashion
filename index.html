A Colorblind Approach to Fashion Object Recognition

What if we couldn't see color? How would we, for example, classify fashion objects solely on the basis of their spatial composition?

This project uses k-means, a statistical model to classify objects from the Metropolitan Museum of Art's 7.6k Costume Institute items. The model was built using machine learning, as an unsupervised learning task.
